= Create shapes and text =

:task:Create simple shapes such as rectangles, polygons, circles, and triangles:
    # In the compositor network editor,
      create a [Shape node|Node:cop2/shape].

    # Use the parameters in the node's parameter editor to choose
      the type of shape, number of points, radius, etc.

:task:Create complex and free-form spline shapes:
    In the compositor network editor,
    create a [Rotoshape node|Node:cop2/rotoshape].

    The [online help for Rotoshape|Node:cop2/rotoshape] has
    extensive information on how to use it to create spline
    shapes.

:task:Create text:
    # Use the [Font operator|Node:cop2/font] to create an image
      containing text.

    # Type the text to render into the __Text__ field, or specify a
      __Text file__ from which to get the text.

    # Use the __Text size__ to resize the text.

    # Use the __Color__ parameter on the __Layout__ tab to set the
      color of the text.
