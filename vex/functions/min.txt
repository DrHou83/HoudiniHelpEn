= min =

#type: vex
#context: all
#tags: math
#group: math


:usage: `int min(int value1, int value2, ...)`
:usage: `float min(float value1, float value2, ...)`

    Returns the minimum of the arguments.

:usage: `<vector> min(<vector> value1, <vector> value2, ...)`

    Returns a vector where each component is the minimum of the corresponding components in the arguments.

:usage: `<type> min(<type> values[])`

    Returns the minimum value in the array.

:usage: `float min(<vector> v)`

    Returns the minimum of the vector's components.

:usage: `<type> min(<type> value)`:
    Returns the argument.
