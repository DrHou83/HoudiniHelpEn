"""Interpolates a value across the currently shaded micropolygon."""

:usage: `float interpolate(float val, float sx, float sy)`
:usage: `vector interpolate(vector val, float sx, float sy)`
:usage: `vector4 interpolate(vector4 val, float sx, float sy)`
:usage: `bsdf interpolate(bsdf val, float sx, float sy)`

These operations can be used to generate antialiasing positions within
the micropolygon rendering engine.

`sx` and `sy` are random values, such as generated by [Vex:nextsample]. Different
values of `sx` and `sy` translate into different random positions on the micropolygon.

The returned value is undefined in the raytracing engine.

{{{
#!vex
vector hitP = interpolate(P, sx, sy);
}}}
