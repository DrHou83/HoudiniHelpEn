"""Computes the rotation matrix or quaternion which rotates the vector a onto the vector b."""

:usage: `matrix3 dihedral(vector a, vector b)`

Computes the rotation matrix which rotates the vector <<a>> onto the vector <<b>>.

:usage: `vector4 dihedral(vector a, vector b)`

Computes the quaternion which rotates the vector <<a>> onto the vector <<b>>.
