= Python Panel =

#helpid: h.pane.pythonpanel

== Overview ==

"""The Python Panel pane allows you to embed custom PySide2 or PyQt5 interfaces into Houdini pane tabs."""

[Image:/images/ui/python_panel_anno.png]

== How to == (howto)

:task:Display a PySide2/PyQt5 interface in the panel:
    Select an entry from the interfaces menu.

:task:Create a new PySide2/PyQt5 interface:
    Click [Icon:BUTTONS/gear] and select __New Interface__.

:task:Edit the current PySide2/PyQt5 interface:
    Click [Icon:BUTTONS/gear] and select __Edit Interface__.

:task:Delete PySide2/PyQt5 interfaces:
    Click [Icon:BUTTONS/gear] and select __Delete Interfaces__.

:task:Edit the interfaces menu:
    Click [Icon:BUTTONS/gear] and select __Edit Menu__.

:task:Reload the current interface:
    Click [Icon:BUTTONS/reload].

== Additional Help ==
Read the [Python Panel Editor|/ref/windows/pythonpaneleditor] help page for information on how to create PySide2 or PyQt5 interfaces and for coding examples.
