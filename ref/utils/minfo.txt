
= minfo =

"""Prints information about movie files."""

== Synopsis ==

    {{{
    minfo filename [filename ...] 
    }}}
== Description ==

    minfo is a command that displays information about movie files. The
    supported formats are:

    * AVI (.avi)

    * MPEG (.mpeg)

    * Houdini HMV format (.hmv)

    * Quicktime (.qt)

    * SGI Movie (.mv)

== Options ==

    For each filename supplied, minfo will display the resolution,
    length and other movie related parameters as appropriate for the
    type of movie (for example, compression, image pixel format,
    blocking, etc.).
