= Extended Operator Info window =

This window provides all of the information found normally by clicking ((MMB)) a node. The Extended Operator Info window also provides information about different attributes of the node, such as type and size. It currently applies only to SOP nodes.

To access this window, click ((RMB)) on a node and choose __Extended Information__.

For surface operator subnetworks, this window also provides information on which node is used as the output.
