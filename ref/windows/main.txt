= Main window =

== Menu bar ==

    Menus:
        See the  [menu reference|/ref/menus].
    Auto Takes:
        Automatically adds parameters to the current take when you change them.
    Takes menu:
        Shows the current take. Choosing New Take from the drop-down menu creates a new take.
    Question mark:
        Shows or hides a help pane.
        
== Status line ==

    At the bottom of the window, the _status line_ displays brief
    messages and prompts you on how to use tools. It also lists quick
    help information on certain interface elements when you move your
    mouse over them.
    
    Cook state and Update buttons:
        See [cooking|/basics/cooking].
