= Animatable Parameters window =

This window allows you to key parameters of a node. To access this window, click ((RMB)) a node.

C:
    Clicking the [Icon:BUTTONS/chooser_node] icon creates a channel for the corresponding parameter.

Parameter:
    The name of the parameter.

K:
    This column sets a key for the corresponding parameter. You can only turn keys on and off for parameters that have channels.
