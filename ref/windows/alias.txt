#helpid: h.menu.settings.aliasvar

= Aliases and Variables window =

This window lets you:

* Create, edit, and delete command aliases.

* Define environment variables.

* Define and load custom expression functions.

== Aliases ==

    This tab lists command aliases available in the textport.

    :task:Edit an alias:
        * Click the alias and edit the definition in the Expansion
          box.

    :task:Create a new alias:
        # Type the alias name in the Alias box.

        # Type the definition in the Expansion box.

        # Press ((Enter)).

    :task:Delete an alias:
        * Click the alias and press ((Del)).

== Variables ==

    This tab lists all environment variables currently defined in
    Houdini.

    :task:Edit a variable:
        * Click the variable and edit the definition in the Value
          box.

    :task:Define a new variable:
        # Type the variable name in the Alias box.

        # Type the value in the Value box.

        # Press ((Enter)).

    :task:Delete a variable:
        * Click the variable and press ((Del)).

== Expressions ==

    This tab lists currently defined  [custom expression
    functions|/expressions/_custom] . This lets you define new,
    higher-level functions from the functions provided with Houdini.

    :task:Edit a function:
        # Click the function in the list on the left.

        # Edit the definition.

        # Click __Apply Changes__.

    :task:Create a new function:
        # Type the variable name in the name box at the bottom.

        # Press ((Enter))

        # Edit the definition.

        # Click __Apply Changes__.

    :task:Edit a function in my default editor:
        # Click the function in the list on the left.

        # Click Edit.

        # Edit the function, save, and exit your editor.

    :task:Delete a function:
        * Click the function in the list on the left and press
          ((Del)).

Textport button:
    Opens a textport window.
Execute button:
    Runs the selected command in the Aliases tab.
