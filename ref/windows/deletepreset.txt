#helpid: h.pane.parm.deletepreset

= Delete preset window =

This window allows you to delete the preset parameters of a node. To access this window, click the [Icon:BUTTONS/gear] gear button in the [parameter editor|/ref/panes/parms] of a node and select __Delete Preset__.

Preset Name:
    Choose the name of the preset you want to delete. All saved presets for that node will appear in the drop-down menu.
    
Delete From:
    The path where the preset is saved.

Accept:
    Deletes the preset.
    
Cancel:
    Closes the window without deleting the preset.
