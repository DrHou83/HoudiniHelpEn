#helpid: h.opmenu.viewdeps

= View dependencies window =

Allows you to view dependencies of selected nodes.

View:
    Dependents:
        Shows nodes that are dependent on the contents or output of the selected node.
    
    References:
        Shows the nodes that the selected node is dependent on.
    
Filter:
    Allows you to filter the dependents/referents.

Expression:
    Shows the expression for the currently selected node that contains the dependency.
