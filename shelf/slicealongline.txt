= Slice along line =

#icon: DOP/slicealongline
#tags: particlefluids

"""Slices a particle fluid using a set number of slicing planes evenly spaced along a certain direction."""

See [how to distribute a particle fluid simulation|/dyno/distribute] for more information.

== Using Slice == (includeme)

    # Click the [Icon:DOP/slicealongline] Slice along line tool on the __Particle fluids__ tab.
    
    # Select the particle fluid you want to slice, or select a previous slice to create sub-slices, then press ((Enter)). See the [help on the Slice tool|slice] for information about sub-slicing regions.
    
    # Use the handle in the viewer or the parameters in the parameter editor to position and rotate the line direction. Drag the "contentric ring" handles to resize the sliced range. Use the __Number of regions__ parameter to set the number of slices made across the line.
    
    [Image:/images/shelf/slice.png]

For specific parameter help see the [Slice along line|Node:dop/slicealongline] node help.
