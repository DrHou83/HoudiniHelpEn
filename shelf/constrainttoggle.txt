= Toggle Constraints =

"""Toggles Constraints on selected objects."""

This tool will enable or disable the constraints on selected objects.

== Using Toggles Constraint ==
             
    # Select one or more objects. Hold ((Shift)) to select multiple objects.

    # Click the [Icon:DATATYPES/boolean] Toggle Constraints tool on the __Constraints__ tab.

This tool will toggle on and off the Enable Constraints parameter on the selected object nodes.
