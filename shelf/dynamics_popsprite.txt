= Sprite Particles =

"""Sets the sprite display for particles."""

The POP Sprite node adds attributes to specify the sprite display of particles. The default is a smoke puff, which you can change on the [POP Sprite|Node:dop/popsprite] node.

TIP:
        You can change the __Mode__ to __Sprite Sheet__ on the [POP Sprite|Node:dop/popsprite] node to use a tiled array that you can create using the [Mosaic COP|Node:cop2/mosaic].
