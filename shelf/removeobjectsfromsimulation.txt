= Remove Objects from Simulation =

"""Takes objects out of the simulation"""

Removes objects from simulation at the first frame, as if it was never simulated.

== Using Remove Objects from Simulation ==

# Select the dynamic objects to remove from simulation. 

# Click the [Icon:SHELF/remove_from_simulation]Remove Objects from Simulation tool on the __Drive Simulation__ tab.

[Image:/images/shelf/remove_objects_from_simulation.png]

In this example, the [torus|/shelf/torus] has been removed from the simulation and the [sphere|/shelf/sphere] is still simulated.
