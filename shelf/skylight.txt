= Sky Light =

#icon: OBJ/light_sky
#tags: lightsandcameras

"""Adds an environment light with an automatic sky reflection map."""

This tool creates an [Environment light|Node:obj/envlight] node with the Sky Environment Map enabled.

To change the sky parameters, click the __Light__ tab and then the __Sky Environment Map__ sub-tab. See the help for the [Sky Environment Map compositing node|Node:cop2/skyenvmap]
for how to use the sky parameters.
