"""Create a normal map from a grayscale image."""

== Overview ==

Create a normal map from a grayscale image.

NOTE:
    This is yields similar results as the Gradient Node, but it is entirely open as a tool building example.

@parameters

Strength:
    Strength of the final normal map. The higher the value, the more raised the values are.
    
Blur Strength:
    Optional blur. Allows for a softening of the edges.
    
Flip X: 
    Flips the Red Channel
    
Flip Y:
    Flips the Green Channel 

@related

* [Color COP|Node:cop/color]
