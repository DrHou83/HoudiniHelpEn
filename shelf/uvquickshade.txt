= UV Quick Shade =

"""Applies an image file as a textured shader to a surface."""

UV Quick Shade lets you quickly put an image file on a surface without creating a [Icon:SHOP/vopmaterial][Material|/shade/materials] or adding UVs. If the input surface(s) do not have UVs, this node adds them using an [orthographic projection|/shelf/uvproject].

== Using UV Quick Shade == (includeme)
    
    # Select the primitives to apply [material|/shade/materials] to.
    
    # Click the [Icon:SOP/uvquickshade][UV Quick Shade|Node:sop/uvquickshade] tool on the __Texture__ tab.

For specific parameter help see the [Icon:SOP/uvquickshade][UV Quick Shade|Node:sop/uvquickshade] node help.
