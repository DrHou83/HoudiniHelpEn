= Add Muscle to Solution =

"""Add one or more muscle objects to an existing metacapture solution."""

This tool is used to insert [muscles|Node:obj/muscle] into an existing metacapture solution.  
If a [capture mirror|Node:sop/capturemirror] node is present, it will attempt to do mirroring based on
the muscle node names.

== Using Add Muscle == (includeme)
    
    # Click the [Icon:SOP/capture]Add Muscle To Solution tool on the __Characters__ tab.
    
    # Select skin to modify the solution of and press ((Enter)) to confirm your selection.
    
    # Select the [muscle objects|Node:obj/muscle] to add to the solution and press ((Enter)) to confirm your selection.
