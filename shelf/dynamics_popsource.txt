= Source =

"""Emits particles from an object in all directions."""

This tool emits particles from a designated object. It can be used to simulate things such a comet emitting debris.

== Using Source == (includeme)

        # Create an object.
        
        # Click the [Icon:POP/source][Source|Node:dop/popsource] tool on the __Particles__ tab. 
        
                The [Icon:SOP/particle][particle system|/dopparticles/] will attach itself to the object you have selected.
        
        # Click [Icon:PLAYBAR/play_forward] [play|/anim/playbar] to see the particles.
        
        [Image:/images/shelf/popsource.png]

For specific parameter help, see the [Icon:POP/source][Source|Node:dop/popsource] node help.
