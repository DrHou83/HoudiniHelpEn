= Wispy smoke =

#icon: SHELF/smoke_wispy
#tags: pyro

"""Creates thin smolder-like smoke rising from the selected object."""

[Image:/images/dynamics/pyro_preset_wispy.png]

See [how to use the pyro tools|/pyro/pyro].

== Tips ==

* You will usually set up other objects, such as pumps, fans, grills, etc. for the smoke to interact with.

* Use the turbulence controls on the `pyrosolver` node, __Simulation__ / __Turbulence__ tab, to set the basic the look of the smoke, then use the up-res to add detail.
