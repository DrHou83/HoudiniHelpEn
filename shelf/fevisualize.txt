"""Creates a finite-element visualization node."""

This tool creates a [FE Visualization node|Node:dop/fevisualization] for a selected finite-element object.

It prepares the selected finite-element object such that it generates all the output attributes that are required by the visualization node.

# Select the geometry to visualize.

# On the __Solid__ tab, click the [Icon:SHELF/visualizefiniteelements] FE Visualization tool.

For parameter help, see the [FE Visualization|Node:sop/fevisualization] node help.
