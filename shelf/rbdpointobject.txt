= RBD Instanced Objects =

"""Creates a rigid body at each point of some source geometry, similarly to how the Copy SOP copies geometry onto points."""

== Using RBD Instanced Objects == (includeme)

    # Select the geometry whose points will be used for instancing.

    # Click the [Icon:DOP/rbdpointobject][Shelf:rbdpointobject] tool on the __Rigid Bodies__ tab.

    # Select the geometry to instance at each point.

    [Image:/images/shelf/rbdpointobject.png]

For specific parameter help see the [Icon:DOP/rbdpackedobject][Node:dop/rbdpackedobject] help.

== Switching to the dynamics level ==

# Click the [Icon:TOOLS/select_dynamics]Dynamics Select Mode button on the [selector and handle controls toolbar|/basics/select].

# Click ((LMB)) the RBD fracture object in the [scene view|/ref/views/3dview].
