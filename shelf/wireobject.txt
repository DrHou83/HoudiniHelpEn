= Wire Object =

"""Creates a Wire Object from SOP Geometry."""

The wire object is a flexible dynamic object that deforms naturally when it interacts with other objects or forces. This tool is useful for simulating grass, [fur|/fur/], and other similar dynamic objects.

== Using Wire Object == (includeme)
        
    # Select the objects to convert to wire objects and press ((Enter)) to confirm your selection.
    
    # Click the [Icon:DOP/wireobject][Wire Object|Node:dop/wireobject] tool from the __Wires__ tab.
 
    [Anim:/images/shelf/wire_object.gif]

For specific parameter help see the [Icon:DOP/wireobject][Wire Object|Node:dop/wireobject] node help.
