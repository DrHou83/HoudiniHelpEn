= Constraint Network =

"""Install a new constraints network or jump to the existing one."""

== Using Constraint Network ==

    # Select one or more objects. Hold ((Shift)) to select multiple objects.

    # Click the [Icon:NETWORKS_chop] Constraint Network tool on the __Constraints__ tab.

This tool creates a new `constraints` CHOP Network inside the object, or reuses an existing one.
