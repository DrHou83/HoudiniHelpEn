= Make Loop =

#tags: games
#icon: SOP/foreach

"""Make a simulation loopable."""

== Overview ==

Allows you to loop an animation on a mesh with the same topology or a volume. 

It can be used to make flames or cloth loop. 

@parameters

Start Frame: 
    Frame in which to start the loop process.
    
End Frame: 
    Final Frame of the loop.
    
Number Loops:
    Number of times the animation will loop within the specified time.
