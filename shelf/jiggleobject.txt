= Jiggle =

#icon:CHOP/jiggle
#tags: animation

"""Jiggles the position of an object based on the animation of another object."""

This tool sets up a [parameter effect|/network/motioneffects] that makes one object follow another, adding overshoot and jiggle as if the object is attached by an elastic.

== Using Jiggle == (includeme)
     
# Select the object that will jiggle.
              
# Click the [Icon:CHOP/jiggle] [Jiggle|Node:chop/jiggle] tool on the __Constraints__ tab of the shelf.

# Select the object to follow and press ((Enter)).

    The parameters of the [Jiggle CHOP|Node:chop/jiggle] will pop up.
