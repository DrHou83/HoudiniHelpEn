"""Drive a camera using a game controller device in first person perspective."""

This is used to constrain an object (camera) to be driven via a game controller 
device in a first person perspective.

== Using the Gamepad Camera ==

    The gamepad camera is made on top of the original camera. Therefore, their
    usages are similar. Please view the documentation of [camera|Node:obj/cam]
