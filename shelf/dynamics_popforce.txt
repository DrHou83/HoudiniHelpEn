= Force =

"""Applies a directional force to particles."""

This tool uses the [POP Force|Node:dop/popflock] node to move particles directionally. By default, this shelf tool applies gravity to your particle system.


== Using Force == (includeme)

        # Create a particle system along the curve using the [Icon:POP/location][Location|/shelf/dynamics_poplocation] or [Icon:POP/source][Source|/shelf/dynamics_popsource] shelf tools.
        
        # Click the [Icon:POP/force][Force|Node:dop/popforce] tool on the __Particles__ tab.
        
        [Image:/images/shelf/POPforce.png]
        
For specific parameter help, see the [Icon:POP/force][Force|Node:dop/popforce] node help.
