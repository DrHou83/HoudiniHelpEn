= Wind Force =

"""Applies forces to resist the current motion of objects relative to a turbulent wind."""

== Using Wind Force == (includeme)
    
    # Select the dynamic object to apply wind force to.

    # Click the [Icon:DOP/windforce][Wind Force|Node:dop/windforce] tool on the __Drive Simulation__ tab.
        
    [Image:/images/shelf/windforce.png]
     
    NOTE:
        If the wind does not apply velocity, increase the __Scale Force__ value on the __Wind__ tab of the [Icon:DOP/windforce][Wind Force|Node:dop/windforce] node in the [parameter editor|/ref/panes/parms].


For specific parameter node help see the [Icon:DOP/windforce][Wind Force|Node:dop/magnetforce] help.
