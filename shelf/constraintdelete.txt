= Delete Constraints =

"""Delete Constraints on selected objects."""

This tool will delete the constraints networks on selected objects.

== Using Delete Constraints ==
             
    # Select one or more objects. Hold ((Shift)) to select multiple objects.

    # Click the [Icon:SOP/delete] Delete Constraints tool on the __Constraints__ tab.

This tool will delete the Constraints networks on the selected object nodes and 
reset the __Constraints Path__ parameter on object nodes to an empty string.
