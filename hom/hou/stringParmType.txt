"""Enumeration of string parameter types."""

A [Hom:hou.StringParmTemplate] is set to one of these types to specify
whether a string parameter will hold an arbitrary string, a reference to
a file, a reference to a node, or a reference to multiple nodes.


@values

:: Regular
:: FileReference
:: NodeReference
:: NodeReferenceList
