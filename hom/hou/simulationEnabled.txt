#type: homfunction
#cppname: hom::simulationEnabled
#group: dynamics

:usage: `simulationEnabled()` -> `bool`

Returns `True` if Houdini is currently set to update simulations. The user can choose to disable simulation updates to improve interactivity using the simulation menu in the bottom right corner of the main window. Use [Hom:hou.setSimulationEnabled] to enable or disable simulation updates programmatically.
