= Hotkey quick reference =

"""Reference list of default Houdini shortcut keys."""

== General keys ==

((Tab)) |
    Open the [tab menu|/basics/tabmenu] of available nodes.

== View keys ==

TIP:
	#glyph: fa-keyboard-o
	
    Hold ((Space)) or ((Alt)) to enter view mode temporarily.

((Space + LMB)) |
    [Icon:TOOLS/tumble]
    Tumble

((Space + RMB)), ((mouse_wheel)) |
    [Icon:TOOLS/dolly]
    Dolly

((Space + MMB)) |
    [Icon:TOOLS/track]
    Track

((Space + Ctrl + LMB)) |
    [Icon:TOOLS/tilt]
    Tilt the camera

((Space + Ctrl + RMB)) |
    [Icon:TOOLS/zoom]
    Zoom the camera lens

((Ctrl + Alt + LMB)) |
    Box zoom

((Space + Z)) |
    Center view tumbling on the point under the pointer.

((Space + H)) |
    Show the default view.

((Space + A)) |
    Zoom the view to show all objects.

((Space + G)) |
    Zoom the view to show all _selected_ objects/geometry.
    
When the pointer is over a viewport:

((W)) |
    Switch shaded/wireframe

((D)) |
    Show display options window

((Space + B)) |
    Switch single/quad viewports

((Space + O)) |
    Switch orthographic/perspective

((Space + E)) |
    Switch view one/view all

((Space + 1)) |
    Switch current viewport to Perspective

((Space + 2)) |
    Top

((Space + 3)) |
    Front

((Space + 4)) |
    Side

((Space + 5)) |
    UV editor

== Tools ==

((S)) |
    [Icon:TOOLS/select]
    [Icon:TOOLS/select_component]
    Select

    Right click this icon for a menu of selection options.
((T)) |
    [Icon:TOOLS/move]
    Move
((R)) |
    [Icon:TOOLS/rotate]
    Rotate
((E)) |
    [Icon:TOOLS/scale]
    Scale
((Enter)) |
    [Icon:TOOLS/handles]
    Node-specific handles
((Esc)) |
    [Icon:TOOLS/view]
    View

== Selection keys ==

((F8)) |
    Toggle object/component selection
((1)) |
    [Icon:TOOLS/select_objects]
    Select objects
((2)) |
    [Icon:TOOLS/select_points]
    Select points
((3)) |
    [Icon:TOOLS/select_edges]
    Select edges
((4)) |
    [Icon:TOOLS/select_faces]
    Select primitives (faces)
((5)) |
    [Icon:TOOLS/select_vertices]
    Select vertices
((6)) |
    [Icon:TOOLS/select_particles]
    Select particles
((7)) |
    [Icon:TOOLS/select_dynamics]
    Select dynamics

When selecting:

((LMB)) |
    Select
((Ctrl + Shift + LMB)) |
    Add to selection
((Ctrl + LMB)) |
    Remove from selection
((Shift + LMB)) |
    Toggle selection
((N)) |
    Select all
((Shift + N)) |
    Select nothing

== Animation ==

((K)) |
    Set keyframe
((Alt + K)) |
    Toggle auto-keying
((Ctrl + K)) |
    Key handle
((Shift + T)) |
    Key translations
((Shift + R)) |
    Key rotations
((Shift + E)) |
    Key scale

== Timeline ==

((Up)) |
    [Icon:PLAYBAR/play_forward]
    Play forward

((Down)) |
    [Icon:PLAYBAR/play_backward]
    Play backward

((Ctrl + Up)) |
    [Icon:PLAYBAR/frame_first]
    First frame

((Left)) |
    [Icon:PLAYBAR/frame_prev]
    Previous frame

((Right)) |
    [Icon:PLAYBAR/frame_next]
    Next frame

((Ctrl + Left)) |
    [Icon:PLAYBAR/frame_first]
    Previous scoped keyframe

((Ctrl + Right)) |
    [Icon:PLAYBAR/frame_first]
    Next scoped keyframe

== Panes ==

((Alt + 1)) |
    Scene view
((Alt + 2)) |
    Network editor
((Alt + 3)) |
    Parameters
((Alt + 4)) |
    Tree view
((Alt + 5)) |
    Textport
((Alt + 6)) |
    Animation editor
((Alt + 7)) |
    Material palette
((Alt + 8)) |
    Details view
((Alt + 9)) |
    Scene view
((Ctrl + T)) |
    New tab
((Alt + Shift + C)) |
    "Tear off" tab into a new floating window.
((Ctrl + PgUp)) |
    Previous tab
((Ctrl + PgDn)) |
    Next tab
((Ctrl + W)) |
    Close tab
((Alt + /)) |
    Close all tabs
        
== Editing the hotkeys ==

See [how to edit the hotkeys|/basics/hotkeys].

TIP:
    ((Ctrl + Alt + Shift + LMB)) click an item in the user interface to open it in the hotkey editor.
