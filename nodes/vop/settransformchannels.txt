= Set Channel Tranform =

"""Sets a transform value when evaluating a __Channel VOP__ in Tranlate/Rotate/Scale mode."""

This operator sets a transform value for the current evaluated channels.

The transform can be set as a matrix or as __Translate__, __Rotate__, __Scale__ vectors.
