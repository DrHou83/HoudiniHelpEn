"""Pixar's prman factory shader"""

This SHOP provides an interface to the shader defined in the prman shader library.  Please see the RSL code shipped with your RenderMan distribution.

NOTE:
    To change the geometry bounds, add the __Min Bounds__ and __Max Bounds__ rendering properties in the [Edit Parameter Interface|/ref/windows/edit_parameter_interface].
