= Auto Bone Chain Interface =

"""The Auto Bone Chain Interface is created by the IK from Objects and IK from Bones tools on the Rigging shelf."""

The Auto Bone Chain Interface asset is created by the [IK from Objects|/shelf/rigging_ikfromobjects] and [IK from Bones|/shelf/rigging_ikfrombones] tools on the __Rigging__ shelf.

@parameters

Match FK to IK:
    Match the FK parameters to the bone IK solution.
    
Match IK to FK:
    Match the IK parameters to the bone FK solution.
    
FK IK Blend:
    Blends between the IK and FK solutions.
    
Stretch Ratio:
    Determines the amount of stretch allowed. 0 for no stretch, up to one for full stretch.
    
FK Control Size:
    The size of the FK controls.
    
IK Control Size:
    The size of the IK controls.
    
End Affector Color:
    The color of the End Affector.
    
FK Controls Color:
    The color of the FK controls.
    
Reposition FK Controls:
    Resets the FK Controls to match the IK solution.
