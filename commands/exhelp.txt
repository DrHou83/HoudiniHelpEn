#type: hscript

= exhelp =

"""Shows usage information for an expression."""

:usage: `exhelp [-k] [<<pattern>>]`

@options
    `-k`:
        Search all command help text for the pattern.
    ` <<pattern>> `:
        Prints help for expressions matching the pattern. If the pattern
        matches an expression exactly, only help for that expression is
        printed.

@related
    - [help|Cmd:help]
