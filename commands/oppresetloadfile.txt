#type: hscript

= oppresetloadfile =

"""Loads channels and parameter values into an operator from a .preset
file."""

:usage: `oppresetloadfile <<operator_name>> <<preset_filename>> `

Once you have a .preset file saved using oppresetsavefile, you can use
this command to apply the preset stored in the file to an operator of
the same type.
