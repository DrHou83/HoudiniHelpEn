#type: hscript

= opspareds =

"""Sets the dialog script describing spare parameters on one or more
nodes."""

:usage: `opspareds <<"parmdesc">> <<nodes>> ... `


TIP:
    This command is generally only used internally by Houdini. For basic
    manipulation of spare parameters, use the  [spare parameters
    editor|/ref/panes/parms]  or the [opspare|Cmd:opspare] command.

`opspareds` lets you describe [spare parameters|/ref/panes/parms]  at
a lower level, using the _dialog script_ format. While this is generally
only used internally, it is possible for you to view the dialog script
generated by Houdini for a spare parameter (using the
[opscript|Cmd:opscript] command with the -p option), edit it by hand,
and insert the edited parameter description into a node or nodes using
this command.



@examples
    {{{
    #!hscript
    opspareds 'parm { name spareparm label "Spare Parm 1" type vector }' geo1
    }}}
    Adds the parameter described by the dialog script (in single quotes)
    to the `geo1` node.


@related
    - [opspare|Cmd:opspare]
    - [opscript|Cmd:opscript]
