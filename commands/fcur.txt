= fcur =

"""Gets or sets the current frame number."""

:usage: `fcur [<<frame>>]`

If no frame is specified, prints the current frame. Otherwise, sets the
current frame to the specified frame.



@options
    `<<frame>>`:
        Sets the current frame.
