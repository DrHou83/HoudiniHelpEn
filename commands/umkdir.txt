#type: hscript

= umkdir =

"""Creates a new directory, <<dir_name>>, in the file system."""

:usage: `umkdir [-p] <<dir_name>>`

@options
    `-p`:
        Make parent directories as necessary, with no failure if the
    specified directory already exists.

    - [ucd|Cmd:ucd]
    - [upwd|Cmd:upwd]
