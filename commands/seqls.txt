#type: hscript

= seqls =

"""Lists the sequences currently loaded in Mplay."""

:usage: `seqls [-f]`

@options
    `-f`:

Lists the file pattern of each image sequence loaded into MPlay. If -f
is used, the full path and file pattern is printed instead.

The file patterns are printed to the textport, one per line.

NOTE:
    This command is only available in MPlay.
