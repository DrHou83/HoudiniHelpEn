= opbfilters =

"""Lists all valid filters for bundles."""

:usage: `opbfilters`

Prints bundle filters in two columns. The first column specifies the
bundle name and the second column provides a description of operators
accepted by the filter.

@related
    - [opbop|Cmd:opbop]
    - [opbadd|Cmd:opbadd]
    - [opbrm|Cmd:opbrm]
    - [opbname|Cmd:opbname]
