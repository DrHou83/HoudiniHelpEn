#type: hscript

= oppresetsavefile =

"""Saves channels and parameter values from an operator to a .preset
file."""

:usage: `oppresetsavefile <<op_name>> <<preset_filename>> `

This command creates/overwrites a .preset file using the channels and
parameter values of the operator you specify. You can then apply that
preset to other operators of the same type using the oppresetloadfile
command.



@related

    - [oppresetls|Cmd:oppresetls]
    - [oppresetrm|Cmd:oppresetrm]
