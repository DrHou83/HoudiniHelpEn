#type: hscript

= opcolor =

"""Gets or sets an operator's tile color in the network editor."""

:usage: `opcolor [-c <<r>> <<g>> <<b>>] <<node>> [<<node2>> ...]`

If you don't specify a color with -c, the tile's current color is
printed.



@options
    `-c <<r>> <<g>> <<b>> `:
        Specify a color using red, green, and blue values between 0 and
        1.
