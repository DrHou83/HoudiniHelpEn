= cophasmeta =

"""Tests if metadata exists on a compositing node."""

:usage: `cophasmeta(<<compositing_node>>, <<metadata_name>>)`

Metadata can be added to sequences with a [Metadata COP|Node:cop2/metadata].
This expression function queries if the metadata with `metadata_name` exists on the COP node `composting_node`.
