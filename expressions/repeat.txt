= repeat =

"""Channel segment function: repeats animation from previous frames."""

:usage: `repeat(<<f1>>, <<f2>>)`

Repeats the animation from frames <<f1>> to <<f2>>.

If you wish to repeat the animation cumulatively,
use the [Exp:cycle] function instead.
