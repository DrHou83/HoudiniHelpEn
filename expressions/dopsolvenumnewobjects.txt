"""Script solver support function."""

:usage: `dopsolvenumnewobjects()`

Returns the number of objects being initialized by the script
solver that is currently running.

This is one of a set of expression functions for use by the [Script
Solver DOP|Node:dop/scriptsolver].  See [Exp:dopsolveobject] and
[Exp:dopsolvenumobjects] for more information.
