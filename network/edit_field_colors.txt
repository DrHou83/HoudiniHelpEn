#type: include

Background color ||
    Meaning ||

Light Green |
    The parameter is controlled by an
    expression, and the current frame is a
    keyframe.

Cyan |
    The parameter is controlled by an
    expression but the current frame is _not_
    a keyframe (or the channel value at the current frame is
    disabled from the
    [dopesheet|/ref/panes/dopesheet]).

Yellow |
    Pending keyframe (the current value at this frame is
    different from the keyed value, but you have not yet
    keyframed it).

Orange |
    The parameter is currently controlled by a channel
    operator (CHOP).
